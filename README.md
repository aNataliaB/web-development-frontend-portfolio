# Web Development Projects
A comprehensive collection of web applications and interactive components designed, structured, and programmed entirely by myself. This repository tracks my learning progression in web technologies, showcasing a transition from semantic layouts and client-side styling to dynamic logic, database integration, and secure user sessions.

## Live Demonstration
The entire system is fully interactive and hosted live via GitHub Pages:
**[https://anataliab.github.io/web-development/](https://anataliab.github.io/web-development/)**

## Repository Structure & Project Descriptions
Every directory represents an independent milestone in building core modern web competencies:

* ### Module 2: Document Architecture & Semantics
**`projekt2a` (Advanced Typography & Forms):** Structured data presentation utilizing semantic HTML5 tags, comprehensive fieldsets, and custom data entry input parameters.
**`projekt2b` (Image Gallery Asset Management):** Media components optimization, scaling ratios control, and clean graphical document structures.
**`projekt2c` (Interactive Form Submissions):** Complex user input layouts mapping dedicated input types (text, email, radio, checkbox) designed for validation routines.

### Module 3: Visual Presentation & Layout Systems (CSS3)
**`projekt3a`, `projekt3b`, `projekt3c` (CSS Architecture):** Mirroring the structures from Module 2, these projects apply customized CSS styling layers. Features focus on cascade specificity, element box modeling, custom border properties, margins configuration, and color palettes alignment.

### Module 4: Client-Side Logic & Applications
* **`projekt4` (Dynamic JavaScript Calculator):** A functional, event-driven web calculator powered by plain JavaScript (ES6). Features dynamic UI rendering, input stream parsing, validation checkpoints, and execution of runtime arithmetic evaluation loops.

### Module 5: Document Templating & Presentation
* **`projekt5` (Automated Invoice Generator):** A styled, clean template designed to parse transaction variables into an aligned, print-ready semantic invoice layout.

### Module 6: Server-Side Form Processing
* **`projekt6` (PHP Form Handling):** Capturing user input fields securely via server global variables (`$_POST` / `$_GET`), implementing basic backend data sanitization, and generating responsive post-submission server responses.

### Module 7: Relational Database Integration
* **`projekt7` (MySQL Connectivity):** Programmatic integration between backend scripts and relational MySQL tables. Focuses on structured data fetching, dynamic table mapping, and persistent database record tracking.

### Module 8: Authentication & Session Security
* **`projekt8` (Secure Authentication Gate):** A complete user login checkpoint. Features secure server-side verification, native state tracking through PHP sessions (`session_start()`), protected subpages restrictions, and explicit redirection states.

## Core Technology Stack
* **Frontend Core:** HTML5, CSS3, JavaScript 
* **Backend Server-Side:** PHP
* **Database Management:** MySQL / Relational Schemas
* **Concepts:** State Management (Sessions), Form Processing, Responsive Visual Aesthetics, DOM Event Loops.
